# 🏅 Olympics Data Analysis — Excel Project

A data analysis project exploring medal performance across the **Tokyo 2020** and **Beijing 2022** Olympic Games using Microsoft Excel.

---

## 📁 File Structure

| Sheet | Description |
|-------|-------------|
| **Cover Page** | Project title and overview |
| **raw data** | 1,350 medal records across both Olympics |
| **analysis** | Top 10 countries and top 3 athletes per Games |
| **Dashboard** | Pivot table — medal counts by country and year |
| **INSIGHTS** | Key findings and strategic recommendations |

---

## 📊 Dataset Overview

- **Games covered:** Tokyo 2020, Beijing 2022
- **Total records:** 1,350 medal entries
- **Countries represented:** 93 (Tokyo) vs 29 (Beijing)
- **Disciplines:** 61 unique sports
- **Fields:** Discipline, Games, Year, Gender, Medal, Type, Athlete, Country, Country Code

---

## 🔑 Key Insights

1. **China** has the highest gold conversion rate (~46%) — nearly half its medals are gold.
2. **USA** dominates Tokyo (105 medals) but drops sharply in Beijing (25 medals), showing heavy reliance on swimming and athletics.
3. Far fewer countries compete in Winter Games — only 29 nations won medals in Beijing vs 93 in Tokyo.
4. **Norway** demonstrates how a few elite athletes can carry an entire nation's medal tally.
5. Niche sports (judo, wrestling, shooting) have less competition, offering a faster path to medals for emerging nations.
